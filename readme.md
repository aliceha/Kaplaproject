# Kaplaproject

## Requirements
*	Node.js
*	Gulp (optional)

## Installation

```bash
$ npm install
$ yarn install
```

## Development

```bash
$ npm run start
$ yarn start
```

## Serve

```bash
$ npm run start
$ yarn serve
```
