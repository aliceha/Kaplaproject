# Kaplaproject

## Requirements
*	Node.js
*	Gulp (optional)

## Installation

```bash
$ npm install
$ yarn install
```

## Features

* Sass
* Live reloading
* Autoprefixing
* Minifying

## Run

```bash
$ npm run start
$ yarn
```
